# Описание

Шаблонизатор для верстки html макетов.
В проекте используются следующие пакеты:

- webpack
- eslint/stylelint
- pug
- sass
- postcss/autoprefixer
- svg-sprites

## Запуск проекта

- `npm i`
- `npm run dev/build`

## Деплой в текущий репозиторий в ветку gh-pages (Только dist файлы)

- `./deploy.sh`
